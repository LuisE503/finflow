## FinFlow 是什么？

FinFlow 是一款帮助您理解和预测个人财务的 PWA：

- 🏷️ **自动分类** 交易到13个类别
- 📈 **现金流预测** 使用 Holt-Winters 预测未来3个月
- 📊 **交互式可视化** 收支趋势
- 🔒 **隐私保护** — 所有数据保留在您的设备上

## 快速开始

```bash
git clone https://github.com/LuisE503/finflow.git
cd finflow && npx serve site
```

## Was ist FinFlow?

FinFlow ist eine PWA, die Ihnen hilft, Ihre persönlichen Finanzen zu verstehen und vorherzusagen:

- 🏷️ **Automatische Kategorisierung** von Transaktionen in 13 Kategorien
- 📈 **Cashflow-Prognose** für 3 Monate mit Holt-Winters
- 📊 **Interaktive Visualisierungen** Ihrer Ausgaben und Einnahmen
- 🔒 **Datenschutz** — alle Daten bleiben auf Ihrem Gerät

## Schnellstart

```bash
git clone https://github.com/LuisE503/finflow.git
cd finflow && npx serve site
```

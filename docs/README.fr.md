## Qu'est-ce que FinFlow ?

FinFlow est une PWA qui vous aide à comprendre et prévoir vos finances personnelles :

- 🏷️ **Catégorisation automatique** des transactions en 13 catégories
- 📈 **Prévision de trésorerie** sur 3 mois avec Holt-Winters
- 📊 **Visualisations** interactives de vos dépenses et revenus
- 🔒 **Confidentialité** — toutes les données restent sur votre appareil

## Démarrage Rapide

```bash
git clone https://github.com/LuisE503/finflow.git
cd finflow && npx serve site
```

## FinFlow とは？

FinFlow は個人の財務を理解し予測するための PWA です：

- 🏷️ **自動分類** 取引を13カテゴリに分類
- 📈 **キャッシュフロー予測** Holt-Winters法で3ヶ月先を予測
- 📊 **インタラクティブな可視化** 収支トレンド
- 🔒 **プライバシー保護** — すべてのデータはデバイス上に保持

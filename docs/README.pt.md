## O que é o FinFlow?

FinFlow é uma PWA que ajuda você a entender e prever suas finanças pessoais:

- 🏷️ **Categorização automática** de transações em 13 categorias
- 📈 **Previsão de fluxo de caixa** para 3 meses com Holt-Winters
- 📊 **Visualizações** interativas de gastos e receitas
- 🔒 **Privacidade** — todos os dados ficam no seu dispositivo

## Início Rápido

```bash
git clone https://github.com/LuisE503/finflow.git
cd finflow && npx serve site
```
